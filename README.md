## 🛠️ Proyectos destacados

### [Admin-T-SQL](https://github.com/elrepositoriodemichel/Admin-T-SQL)
> Colección de scripts de administración, diagnóstico y utilidades para SQL Server, escritos en T-SQL. Cada utilidad incluye documentación detallada de uso, parámetros e interpretación de resultados.

| Utilidad | Descripción |
|---|---|
| [QueryResourceHistory](https://github.com/elrepositoriodemichel/Admin-T-SQL/tree/main/QueryResourceHistory) | SP para análisis histórico de consumo de recursos desde la caché de planes |
| [ServerBaseline](https://github.com/elrepositoriodemichel/Admin-T-SQL/tree/main/ServerBaseline) | Guía y scripts para establecer una baseline completa de la instancia |
| [Backups](https://github.com/elrepositoriodemichel/Admin-T-SQL/tree/main/Backups) | Guía comparativa SSMS vs T-SQL para gestión de copias de seguridad |
| [OldestActiveTransactions](https://github.com/elrepositoriodemichel/Admin-T-SQL/tree/main/OldestActiveTransactions) | Diagnóstico de transacciones activas antiguas incluyendo sesiones idle |
| [LogFileInfo](https://github.com/elrepositoriodemichel/Admin-T-SQL/tree/main/LogFileInfo) | Información detallada de ficheros de log de todas las bases de datos |
| [GetChainLog](https://github.com/elrepositoriodemichel/Admin-T-SQL/tree/main/GetChainLog) | Cadena de restauración completa a un punto en el tiempo desde msdb |
